Installation How to Install the Template

Install Theme

Here's a brief information about how to install your theme download from scratch

For installing this template you need to make sure that you have the following extensions on your webserver. You will need to contact your webhost if the following requirements are missing.

  • PHP Mail support;
  • JavaScript Enabled Web Browser.

Installing the template is pretty simple. Once you have downloaded the theme files after purchase, extract the downloaded theme file to the local disk on your computer. Inside the extracted folder there will be 2 folders labeled docs and html. Inside the html folder, you will find the template files which you will have to upload to your web server via an FTP client (e.g. Filezilla or Total Commander).

The files can be edited locally and the main file, viewed directly on your web browser from your computer without any internet connection.

  1. For the template to be accessible on the internet, you need to upload the extracted files (of the downloaded zip) to your web server. The server space can be purchased at many different places around the internet. You need somewhere to put the files that you purchased so others can find them online.
  2. In the extracted folder of the downloaded theme, you will find three folders (css, img and js) as well as index.html in the directory. The contents to be modified are found in index.html. The css folder contain all the stylesheets which customize the look and feel of the website. The js file contains all script files required for the various effects. The img directory contains all the images related to the website.
  3. To get the website online, it is advised to make the changes locally from the extracted directory and afterwards upload the said directory (which contains the subdirectory css, img and js and the core file index.html and mail.php) to your web server.
  4. Start your FTP client (I persoanlly use Filezilla which is free) and login. Login to your server. You will need your server name (often times this will be your domain name), username, and password that you setup with the hosting company.
  5. Locate the domain folder you wish to upload your template files to on your server. You are going to want to click and drag every file that you have labeled .html, .css, .php, .js into your FTP client.

Markup Markup customizations

This theme is a based on twitters bootstrap and it is fully responsive.The default Bootstrap grid system utilizes 12 columns, making for a 940px wide container without responsive features enabledand the grid adapts to be 724px and 1170px wide depending on your viewport. Below 767px viewports, the columns become fluid and stack vertically.

Each Section is properly documented and indented to facilitate changes

How To Edit Color

If you would like to edit the color, font, or style of any elements in one of these columns, you would do the following:

.logo h1 a {color: #464646;}

Change Structure

If you find that your new style is not overriding, it is most likely because of a specificity problem. Scroll down in your CSS file and make sure that there isn't a similar style that has more weight.

header .logo h1 a { color: #464646;}

JavaScript Files

The custom.js file is the main file used to launch the different scripts. The different scrips used are:

Tag Description
jquery-2.1.0.min.js jQuery is a Javascript library that greatly reduces the amount of code that you must write.
bootstrap.min.js Packed with many functionalies such as carousel slider, responsivity, tabs, drop downs, buttons, and many other features.
modernizr.min.js Modernizr is an open-source JavaScript library that helps you build the next generation of HTML5 and CSS3-powered websites.
jquery.prettyPhoto.js A plugin for displaying images, html content and multi-media in a "lightbox" that floats overtop of web page.
custom.js Contains all the settings for the different effects used. Apart from being properly commented, further documentation may be found on the above stated websites.

//    Scroll Up
// --------------------------------------------------------     
$(window).scroll(function() {
    if ($(this).scrollTop() > 100) {
        $('.scroll-up').fadeIn();
    } else {
        $('.scroll-up').fadeOut();
    }
});

$('.scroll-up').click(function() {
    $("html, body").animate({
        scrollTop: 0
    }, 600);
    return false;
});

// --------------------------------------------------------
//    Navigation Bar
// --------------------------------------------------------     
$(window).scroll(function(){    
    "use strict";    
    var scroll = $(window).scrollTop();
    if( scroll > 60 ){        
        $(".navbar").addClass("scroll-fixed-navbar");                
    } else {
        $(".navbar").removeClass("scroll-fixed-navbar");
    }
});

// --------------------------------------------------------
//    Smooth Scrolling
// --------------------------------------------------------     
$(".navbar-nav li a[href^='#']").on('click', function(e) {
    e.preventDefault();
    $('html, body').animate({
        scrollTop: $(this.hash).offset().top
    }, 1000);
});

function toggleIcon(e) {
    $(e.target)
        .prev('.panel-heading')
        .find('.panel-title a')
        .toggleClass('active')
        .find("i.fa")
        .toggleClass('fa-plus-square fa-minus-square');
}
$('.panel').on('hidden.bs.collapse', toggleIcon);
$('.panel').on('shown.bs.collapse', toggleIcon);
